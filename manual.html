VidControl Manual

Basic concept

VidControl allows you to configure the default attributes of <video> tags (AKA HTML5 video) on any page, independent of the settings baked-in by the webmaster.

Rulesets

In the options menu for VidControl, you will see a table of "rulesets". These rulesets control which pages to make video-related attribute changes to, and what changes should be made to the videos on those pages. The highest priority ruleset that matches the page URL will be applied.

Adding a ruleset

A new ruleset may be added by clicking the "Add" button. This will prepend a new row onto the rulesets table. You may then fill in the ruleset options as desired. Click "Save" to store your changes. Running tabs and/or windows will need to be reloaded for changes to take effect.

Changing a ruleset

To change a ruleset, just edit the ruleset options, and then click "Save". Running tabs and/or windows will need to be reloaded for changes to take effect.

Ruleset options

regex

A JavaScript regular expression that determines which pages the ruleset may apply to.

autoplay controls loop mute preload

These options merely control attributes of the <video>. Please reference the descriptions of these options on the Mozilla Developer Network for more details.

priority

The priority associated with this ruleset. For a given page, the highest priority ruleset with a regex matching the page URL will be applied.
